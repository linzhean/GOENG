# GOENG
flutter 期末專題

# 功能介紹
1.登錄注冊
2.首頁、對話款新增單字集（左滑右滑）
3.收藏單字卡界面、新增單字集對話框
4.賬號（個人資料）、單子卡列表（解釋説明的列表 )
